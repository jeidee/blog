# 포스트 작성 방법

\_posts 디렉토리에 다음 규칙으로 파일을 생성한 후 마크다운 문법으로 문서를 생성합니다.
파일 커밋 후 master 브랜치에 push하면 자동으로 포스트가 등록됩니다.

다음 사이트에서 생성된 포스트를 확인할 수 있습니다.

https://jeidee.github.io

- 파일명 : YYYY-MM-DD-{{제목}}.md

# 작성한 문서 미리보기

jekyll을 사용하고 있으므로 다음과 같이 실행해서 작성한 문서를 미리볼 수 있습니다.

    $ bundle exec jekyll serve

jekyll의 설치는 다음 사이트를 참고합니다.

https://jekyllrb.com

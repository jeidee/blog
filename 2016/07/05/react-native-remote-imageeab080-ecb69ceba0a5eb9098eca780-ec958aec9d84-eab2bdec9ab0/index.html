react-native remote image가 출력되지 않을 경우

July 5, 2016

ios에서 info.plist를 다음과 같이 수정한다.

  
<key>NSAppTransportSecurity</key>
  
<dict>
  
<key>NSAllowsArbitraryLoads</key>
  
<true/>
  
</dict>

다음 링크를 참고하도록 한다.

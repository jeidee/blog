express view engine으로 swig를 사용해 보자. express로 웹앱의 기본 골격 프로젝트를 생성했다고 가정한다. package.json에 다음과 같이 dependencies 모듈을 추가해 준다. "swig": "1.4.2" app.js에 다음 코드를 추가한다. var swig = require('swig'); app.js에서 view engine 관련된 코드를 다음과 같이 수정한다. // view engine setup //app.set('views', path.join(__dirname, 'views')); //app.set('view engine', 'jade'); app.set('view engine', 'html'); app.set('views', __dirname + '/views'); app.engine('.html', swig.renderFile); cache를 false로 하지 않으면 template html을 수정해도 cache된 template html을 사용하므로 다음과 같이 cache를 false로 설정한다.

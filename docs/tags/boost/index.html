osx + eclipse CDT에서 boost설정

osx에서 boost 설치 $ brew install boost sample source #include <boost/regex.hpp> #include <iostream> #include <string> int main() { std::string line; boost::regex pat( "^Subject: (Re: |Aw: )\*(.\*)" ); while (std::cin) { std::getline(std::cin, line); boost::smatch matches; if (boost::regex_match(line, matches, pat)) std::cout << matches[2] << std::endl; } } 프로젝트 설정 프로젝트의 속성을 연다. C/C++ Build > Settings > Tool Settings > GCC C++ Linker > Libraries를 선택한다. Libraries(-l) 항목에 boost_regex를 입력한다.

virtual box vdi 용량 늘리기

windows 7에 virtual box를 설치하고 xubuntu를 guest os로 설치했다고 가정한다. 저장소는 vmdk일 경우 먼저 vdi로 변경 후 사이즈를 조절하도록 한다. vdi 용량 늘리기 1) vmdk를 vdi로 변경 cmd를 열고 vmdk 파일이 있는 곳으로 이동한다. VBoxManage.exe의 clonehd 기능으로 vmdk파일을 vdi로 변경한다. > "C:Program FilesOracleVirtualBoxVBoxManage.exe" clonehd server1-disk1.vmdk server1-disk1.vdi &#8211;format vdi 2) vdi 사이즈 수정 vdi의 사이즈를 변경하기 위해서 VBoxManage.exe의 modifyhd 기능으로 사이즈를 변경한다. –resize 매개변수의 값은 단위가 MB이며 30GB를 원할 경우 30 * 1024 = 31744를 입력하도록 한다.

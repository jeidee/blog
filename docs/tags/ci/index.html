젠킨스 사용법

우분투에 설치 공식홈 설치 안내를 따라 다음과 같이 설치한다. 1) 젠킨스 설치를 위한 데비안 패키지 리포지토리사용을 위해 다음과 같이 시스템에 키를 추가해 준다. wget -q -O &#8211; http://pkg.jenkins-ci.org/debian/jenkins-ci.org.key | sudo apt-key add &#8211; 2) /etc/apt/sources.list 파일에 다음 내용을 추가해 준다. deb http://pkg.jenkins-ci.org/debian binary/ 2) 젠킨스를 설치한다. sudo apt-get update sudo apt-get install jenkins 실행 설치된 젠킨스를 다음과 같이 시작한다. sudo service jenkins start 젠킨스의 초기 사용 포트는 8080이며 다음과 같이 변경할 수 있다.

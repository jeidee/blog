준비

openssl이 설치되어 있고 ubuntu에서 작업한다고 가정합니다.

설치되어 있지 않을 경우 다음과 같이 설치합니다.

      
sudo apt-get install openssl

개인키를 생성합니다.

      
openssl genrsa 1024 > key.pem

만들어진 개인키를 사용해 cert 인증서 파일을 생성합니다.

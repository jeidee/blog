PAM 라이브러리 설치

      
$ sudo apt-get install libpam0g-dev

expat 라이브러리 설치

$ sudo apt-get install rebar

$ sudo apt-get install libssl-dev

yaml 라이브러리 설치

      
$ sudo apt-get install libyaml-dev

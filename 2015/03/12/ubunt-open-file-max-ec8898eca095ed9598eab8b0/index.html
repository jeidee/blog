ubunt open file max 수정하기

#open file max #ubuntu

test 계정이 있다고 가정하고 다음과 같이 작업합니다.

1) sysctl.conf 수정

  
$ sudo vi /etc/sysctl.conf
  
fs.file-max = 1513687

&#8230;

$ sudo sysctl -p
  
fs.file-max = 1513687

2) limits.conf 수정

  
$ sudo vi /etc/security/limits.conf
  
test soft nofile 100000
  
test hard nofile 100000

3) reboot

  
$ sudo reboot
  
&#8230;
  
$ ulimit -n
  
100000

참고

  • file의 open handle 개수 늘리기

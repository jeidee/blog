    c++ eclipse에서 c++ 11/0x 지원 설정

    먼저 C++ 프로젝트를 생성한다. 생성된 프로젝트에서 Properties 메뉴를 선택한다. C/C++ Build > Settings > Tool Settings > GCC C++ Compiler > Miscellaneous -> Other Flags에 “-std=c++0x”값을 입력한다. C/C++ General > Paths and Symbols > Symbols > GNU C++에서 “Add”버튼을 클릭하고 Name필드에 “GXX_EXPERIMENTAL_CXX0X“값을 입력한다. Value필드는 빈 채로 둔다. Apply버튼을 클릭하고 OK 버튼을 클릭한다. 다시 빌드한다. 참고 eclipse cdt c++11/0x support
